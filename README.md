# SWIFTY_COMPANION

### About

Simple React Native application that retrieve datas from 42 API to display informations about students.

### Techs

React Native
Typescript

## Installation

You must have Android Studio installed and run a virtual device from inside.
`npm install`
`npm start`
`npm run android`
